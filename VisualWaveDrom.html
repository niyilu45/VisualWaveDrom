Vim 模式使用说明

快速开始

开启“Vim 模式”后，单击波形区域使用波形命令，单击 JSON 编辑器使用标准 Vim 文本命令。波形触发的 JSON 高亮不会改变键盘区域。

Esc退出选择、替换、文本编辑或连接操作，返回 NORMAL 数字 + 命令重复执行，例如 5l（数字 5 + 小写 L）向右移动 5 格 ?随时打开本使用说明

模式和生效范围

NORMAL移动、复制、删除及执行命令 VISUAL选择连续波形格或连续信号行 INSERT / REPLACE连续插入波形或连续替换波形 WAVE键盘命令作用于当前编辑波形图 JSONJSON 编辑器使用标准 Vim 的 NORMAL、INSERT 和 VISUAL 模式 未聚焦键盘由当前目录、工具栏或说明编辑控件正常处理

波形格和信号行

h / j / k / l（小写 L）左、下、上、右移动波形光标 w / b / e跳到后一个、前一个波形变化点，或当前/下一波形段末尾 0 / $ / gg / G跳到行首、行尾、首行、末行 v / V进入波形格选择或信号行选择 o在 VISUAL 中交换选择起点和终点 y / p / x复制、覆盖粘贴、删除选中的波形格 r + 波形字符替换当前位置；R 连续替换；| 可直接写入间隔 i / I / A从当前格、当前行首或当前行末进入波形插入模式 t当前选中内容可编辑时，直接打开数据、分组或已选连接线的标签输入框 yy / dd复制或删除当前信号行，复制时不包含 node o / O在当前信号行之后或之前插入空行 [r / ]r上移或下移信号行，可使用数字前缀 Space+g将 VISUAL 中选择的连续信号行建立为分组

分组和连接线

Space+a / Space+c在波形区域中新增连接或选择已有连接线 连接线 t / d编辑或删除当前选中的连接线标签 分组 t / d编辑或删除当前选中的分组标签

历史记录

u / Ctrl+r使用 VisualWaveDrom 的统一历史记录撤销或重做 .重复最近一次可重复的修改 JSON 编辑器单击后进入 JSON Vim；按 i 编辑，按 Esc 返回 NORMAL，支持 u 和 Ctrl+r

命令行

按 : 打开命令行。:open 后只填写目录中的自动编号，不填写标题。

:open 1.2按自动编号打开波形图 :w保存当前波形库 :format格式化并对齐当前 JSON :json / :nav显示或隐藏 JSON 面板、波形目录 :new在当前目录中新建空波形图 :move 编号将当前波形图移动到指定目录 :debug / :noh切换调试模式、清除选择和高亮 :set vim / :set novim开启或关闭 Vim 模式
